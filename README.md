# Synthetic-Opinion-Dataset-Framework

LLMを用いた制御可能かつ多様な人工意見データセットの自動構築手法に関する研究

## 概要
本リポジトリは、LLMを活用して特定の立場やテーマに沿った多様な意見データを自動生成するフレームワークのソースコードと、生成されたデータセットを公開するものです。

## 内容
- `opinion_generation_tool.ipynb`: Google Colabで動作する生成ツール（Web UI付き）
- `synthetic_xstance_300.csv`: 提案手法で生成したx-stance形式の人工データ
- `synthetic_shinagawa_300.csv`: 提案手法で生成した行政アンケート形式の人工データ
